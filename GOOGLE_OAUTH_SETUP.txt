# ========================================
# Google OAuth 2.0 - Настройка для вашего сайта
# ========================================

Ваш сайт: https://smart-schedule-manager.netlify.app

# ========================================
# ШАГ 1: Откройте Google Cloud Console
# ========================================
https://console.cloud.google.com/

# ========================================
# ШАГ 2: APIs & Services → Credentials
# ========================================
Нажмите на ваш OAuth 2.0 Client ID

# ========================================
# ШАГ 3: Authorized JavaScript origins
# ========================================
Добавьте (или замените):

https://smart-schedule-manager.netlify.app

# ========================================
# ШАГ 4: Authorized redirect URIs
# ========================================
Добавьте (или замените):

https://smart-schedule-manager.netlify.app/.netlify/functions/api/auth/google/callback

⚠️ ВАЖНО: URL должен быть ТОЧНО таким, включая /.netlify/functions/api/auth/google/callback

# ========================================
# ШАГ 5: Сохраните
# ========================================
Нажмите кнопку "SAVE"

# ========================================
# ШАГ 6: Скопируйте учетные данные
# ========================================
Client ID: скопируйте в NETLIFY_ENV_VARIABLES.txt
Client Secret: скопируйте в NETLIFY_ENV_VARIABLES.txt

# ========================================
# ГОТОВО!
# ========================================
